# Sistemas Operacionais Derivados em Diferentes Dispositivos

Diversos sistemas operacionais modernos são construídos sobre outros sistemas já existentes. Isso permite aproveitar núcleos estáveis e confiáveis, adicionando camadas especializadas para funções específicas. A seguir estão cinco exemplos de sistemas usados em diferentes tipos de dispositivos.

## Xbox One System Software

O **Xbox One System Software**, utilizado no console **Xbox One** da Microsoft, é derivado do **Microsoft Windows**. A Microsoft adaptou tecnologias do Windows para criar um sistema voltado ao funcionamento de um console de videogame.

A arquitetura do sistema é dividida em três ambientes principais que funcionam simultaneamente:

- um sistema dedicado ao console
- um sistema responsável pelos jogos
- um sistema para aplicativos

Esses ambientes são controlados por um **hipervisor**, que gerencia os recursos do hardware e mantém cada ambiente isolado.

Sua principal vantagem é permitir **multitarefa**, como jogar enquanto downloads ou aplicativos continuam funcionando. Em comparação com outros sistemas de consoles, ele é mais próximo da arquitetura de um computador tradicional. A desvantagem é a **maior complexidade do sistema**, que pode consumir mais recursos.

## Tizen

(https://commons.wikimedia.org/wiki/File:Tizen_Logo.png)

O **Tizen** é um sistema operacional desenvolvido principalmente pela Samsung em parceria com a Linux Foundation. Ele é **derivado do kernel do Linux**.

Nesse sistema, o kernel Linux gerencia o hardware e os processos, enquanto o Tizen fornece a interface e o ambiente de aplicativos. Ele é usado principalmente em:

- Smart TVs
- Smartwatches
- Eletrodomésticos inteligentes
- Dispositivos de Internet das Coisas (IoT)

Sua vantagem é ser **leve e eficiente para dispositivos domésticos**, consumindo poucos recursos. Em comparação com sistemas de consoles, ele não precisa lidar com gráficos complexos ou alto desempenho em jogos. A desvantagem é possuir **um ecossistema menor de aplicativos**.

## Orbis OS

O **Orbis OS** é o sistema operacional do **PlayStation 4**, desenvolvido pela Sony. Ele é **derivado do FreeBSD**, um sistema da família Unix.

A Sony modificou o kernel do FreeBSD para criar um sistema otimizado para o hardware do console. O Orbis OS gerencia:

- execução de jogos
- interface do console
- downloads
- serviços online

Sua maior vantagem é o **alto desempenho para jogos**, já que ele foi projetado para um único hardware específico. Comparado ao sistema do Xbox One, ele tende a ser mais enxuto e direto. A desvantagem é a **menor flexibilidade para executar outros tipos de aplicações**.

## OpenWrt

O **OpenWrt** é um sistema operacional voltado para **roteadores e dispositivos de rede**. Ele também é **derivado do kernel Linux**.

Diferente do firmware padrão que vem nos roteadores, o OpenWrt transforma o dispositivo em um pequeno sistema Linux completo. Isso permite instalar programas adicionais e realizar configurações avançadas de rede.

Entre suas funções estão:

- firewall avançado
- gerenciamento de rede
- criação de VPN
- monitoramento de tráfego

Sua principal vantagem é a **grande liberdade de configuração e controle da rede**. Em comparação com os outros sistemas apresentados, ele é muito mais voltado à infraestrutura de rede. A desvantagem é que **exige maior conhecimento técnico**.

## Horizon OS

O **Horizon OS** é o sistema operacional do **Nintendo Switch**, desenvolvido pela Nintendo. Ele utiliza um kernel baseado em arquitetura **Unix**.

Esse sistema foi projetado para ser leve, seguro e eficiente em energia, algo essencial para um console híbrido que funciona tanto portátil quanto conectado à TV.

O sistema organiza suas funções em serviços separados e executa jogos em ambientes isolados chamados **sandboxes**.

Sua vantagem é a **eficiência e estabilidade**, permitindo iniciar e pausar jogos rapidamente sem consumir muita bateria. Comparado aos sistemas do Xbox One e do PlayStation 4, ele prioriza mais a **economia de energia** do que o desempenho máximo. A desvantagem é ser um sistema **mais fechado e com pouca possibilidade de personalização**.

## Comparação Geral

Os cinco sistemas analisados possuem bases diferentes, mas seguem a mesma ideia de aproveitar tecnologias já existentes.

- **Xbox One System Software** → derivado do Windows e focado em multitarefa no console  
- **Tizen** → derivado do Linux e voltado para dispositivos domésticos inteligentes  
- **Orbis OS** → derivado do FreeBSD e otimizado para jogos no PlayStation 4  
- **OpenWrt** → derivado do Linux e especializado em gerenciamento de redes  
- **Horizon OS** → baseado em arquitetura Unix e projetado para eficiência no Nintendo Switch  

Apesar de terem origens semelhantes em sistemas anteriores, cada um foi adaptado para atender necessidades muito específicas, como entretenimento, redes ou dispositivos domésticos.
