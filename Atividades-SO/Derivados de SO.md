# Sistemas Operacionais Derivados em Diferentes Dispositivos

Diversos sistemas operacionais modernos são construídos sobre outros sistemas já existentes. Isso permite aproveitar núcleos estáveis e confiáveis, adicionando camadas especializadas para funções específicas. A seguir estão cinco exemplos de sistemas usados em diferentes tipos de dispositivos.

---

## Xbox One System Software

![Xbox One](https://upload.wikimedia.org/wikipedia/commons/4/43/Xbox_One_consoles_montage.png)

O **Xbox One System Software**, utilizado no console **Xbox One** da Microsoft, é derivado do **Microsoft Windows**. A Microsoft adaptou tecnologias do Windows para criar um sistema voltado ao funcionamento de um console de videogame. :contentReference[oaicite:0]{index=0}

A arquitetura do sistema é dividida em três ambientes principais que funcionam simultaneamente:

- um sistema dedicado ao console  
- um sistema responsável pelos jogos  
- um sistema para aplicativos  

Esses ambientes são controlados por um **hipervisor**, que gerencia os recursos do hardware e mantém cada ambiente isolado.

Sua principal vantagem é permitir **multitarefa**, como jogar enquanto downloads ou aplicativos continuam funcionando. Em comparação com outros sistemas de consoles, ele é mais próximo da arquitetura de um computador tradicional.

---

## Tizen

![Tizen Logo](https://upload.wikimedia.org/wikipedia/commons/3/3e/Tizen_logo.svg)

O **Tizen** é um sistema operacional desenvolvido principalmente pela Samsung em parceria com a Linux Foundation. Ele é **derivado do kernel do Linux**.

Nesse sistema, o kernel Linux gerencia o hardware e os processos, enquanto o Tizen fornece a interface e o ambiente de aplicativos. Ele é usado principalmente em:

- Smart TVs  
- Smartwatches  
- Eletrodomésticos inteligentes  
- Dispositivos de Internet das Coisas (IoT)

Sua vantagem é ser **leve e eficiente para dispositivos domésticos**, consumindo poucos recursos.

---

## Orbis OS

![PlayStation 4](https://upload.wikimedia.org/wikipedia/commons/5/5e/PS4-Console-wDS4.jpg)

O **Orbis OS** é o sistema operacional do **PlayStation 4**, desenvolvido pela Sony. Ele é **derivado do FreeBSD**, um sistema da família Unix.

A Sony modificou o kernel do FreeBSD para criar um sistema otimizado para o hardware do console. O Orbis OS gerencia:

- execução de jogos  
- interface do console  
- downloads  
- serviços online  

Sua maior vantagem é o **alto desempenho para jogos**, já que ele foi projetado para um único hardware específico.

---

## OpenWrt

![OpenWrt](https://upload.wikimedia.org/wikipedia/commons/0/03/OpenWrt_Logo.svg)

O **OpenWrt** é um sistema operacional voltado para **roteadores e dispositivos de rede**. Ele também é **derivado do kernel Linux**.

Diferente do firmware padrão que vem nos roteadores, o OpenWrt transforma o dispositivo em um pequeno sistema Linux completo. Isso permite instalar programas adicionais e realizar configurações avançadas de rede. :contentReference[oaicite:1]{index=1}

Entre suas funções estão:

- firewall avançado  
- gerenciamento de rede  
- criação de VPN  
- monitoramento de tráfego  

Sua principal vantagem é a **grande liberdade de configuração e controle da rede**.

---

## Horizon OS

![Nintendo Switch](https://upload.wikimedia.org/wikipedia/commons/3/38/Nintendo_Switch_Console_Docked_wJoyConRB.jpg)

O **Horizon OS** é o sistema operacional do **Nintendo Switch**, desenvolvido pela Nintendo. Ele utiliza um kernel baseado em arquitetura **Unix**.

Esse sistema foi projetado para ser leve, seguro e eficiente em energia, algo essencial para um console híbrido que funciona tanto portátil quanto conectado à TV.

O sistema organiza suas funções em serviços separados e executa jogos em ambientes isolados chamados **sandboxes**.

Sua vantagem é a **eficiência e estabilidade**, permitindo iniciar e pausar jogos rapidamente sem consumir muita bateria.

|--------|----------------------|---------------------------|---------------|
| Xbox One System Software | Xbox One | Windows | Microsoft |

## 3. Vantagens e Desvantagens dos Sistemas

| Sistema | Vantagens | Desvantagens |

| Orbis OS | Alto desempenho e otimização para jogos | Sistema fechado e pouca flexibilidade |
| OpenWrt | Grande liberdade de configuração e controle de rede | Exige conhecimento técnico para uso avançado |
